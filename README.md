# Personal Portfolio

A responsive and modern personal portfolio website built to showcase my academic background, technical skills, projects, and professional experience. 

## 🚀 Tech Stack

*   **Framework:** React 
*   **Styling:** Tailwind CSS[cite: 2]
*   **Build Tool:** Vite[cite: 2]

## 📁 Project Structure

The application is structured using a modular component-based architecture to keep the code clean and maintainable. Key directories and files include:

*   `src/components/`: Contains all the isolated UI components[cite: 2].
    *   `Portfolio.jsx`: The main layout orchestrating the page[cite: 2].
    *   `HeaderSection.jsx`: Displays the primary introduction and contact links[cite: 2].
    *   `ProfileSection.jsx`: Houses the professional summary[cite: 2].
    *   `SkillsSection.jsx`: Organizes technical proficiencies and tools[cite: 2].
    *   `ExperienceEducationSection.jsx`: Details work history and academic background[cite: 2].
    *   `ProjectsSection.jsx`: Highlights key academic and ML/AI projects[cite: 2].
    *   `AchievementsSection.jsx`: Lists hackathon successes and certifications[cite: 2].
    *   `LoadingScreen.jsx`: Handles the initial loading state[cite: 2].
*   `tailwind.config.js`: Configuration for custom Tailwind CSS utility classes[cite: 2].
*   `vite.config.js`: Build and development server configuration[cite: 2].

## 🛠️ Getting Started

Follow these steps to run the project locally on your machine.

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed.

### Installation

1.  Clone the repository to your local machine:
    ```bash
    git clone [https://github.com/Abhishek-Hiremath01/Portfolio.git](https://github.com/Abhishek-Hiremath01/Portfolio.git)
    ```
2.  Navigate into the project directory:
    ```bash
    cd Portfolio
    ```
3.  Install the required dependencies[cite: 2]:
    ```bash
    npm install
    ```
4.  Start the Vite development server:
    ```bash
    npm run dev
    ```
5.  Open your browser and navigate to the local host address provided in your terminal (usually `http://localhost:5173`).

## 📝 Customization

To update the portfolio with new achievements or projects, simply navigate to the relevant components inside the `src/components/` directory[cite: 2]. Update the data structures within those files, and the UI will automatically render your latest information.
